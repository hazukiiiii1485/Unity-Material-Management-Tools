〇Unity Asset Library Tool
Unityエディタ拡張として動作する、ゲームに使用した自作素材/フリー素材の一括管理ツールです。

〇機能
- テクスチャ・音声・3Dモデルのスキャン＆一覧表示
- サムネイルプレビュー
- タグ・カテゴリの編集・保存（JSON）
- 名前・種別・タグ・カテゴリでの検索・フィルタ
- MD5ハッシュによる重複検出

〇使用方法
1. `Assets/Editor/AssetLibrary/` 内のスクリプトをUnityプロジェクトに導入
2. Unityメニューの `Tools > Asset Library` を開く
3. 「Scan Project」ボタンでアセットをスキャン

〇動作環境
- Unity 2022以降推奨
- 3D(Built-in Render Pipeline)
